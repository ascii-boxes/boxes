# datagen

Convert boxes config to Jekyll data file for the website's design explorer

### Prerequisites

- Python 3.10 or newer
- Poetry 1.7 or newer

### Setup

In datagen folder:

- `virtualenv .venv`
- `.venv\Scripts\activate.bat` (Windows) or `source .venv/bin/activate` (Unix/Linux)
- `poetry install`
- `poetry run convert --help`
